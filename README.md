# Nextgrowth Backend Exercice

## Usage

```shell
$ # Clone the repo
$ git clone ...

$ # Go to the root folder
$ cd nextgrowth/

$ # Install the required libraries
$ npm install

$ # Set up the project configuration
$ cp .env.example .env
$ # Make sure to update the .env configuration file

$ # Seed the database
$ npm run db:seed

$ # Start the Server
$ npm start
```
